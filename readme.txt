-- Description --
Allows you to block visitors from specific countries or IP Address.

-- Installation --
1. Upload the entire shogo-block-visitor directory to your wordpress plugin directory "/wp-content/plugins"
2. Login to your Wordpress Admin Panel and activate the plugin through the plugins menu. "www.your-wordpress-website.com/wp-admin/plugins.php"
3. A Shogo Forms Link will appear in your wordpress menu at the bottom. Click the link and create form to start.

On Github
1. Clone shogo-block-visitor from https://github.com/shogo23/ to your wordpress plugins directory.
2. Rename directory to "shogo-block-visitor".
3. Login to your Wordpress Admin Panel and activate the plugin through the plugins menu. "www.your-wordpress-website.com/wp-admin/plugins.php"
4. A Shogo's Block Visitor Link will appear in your wordpress menu at the bottom. Click the link and create form to start.
